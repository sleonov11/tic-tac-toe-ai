# Крестики-нолики с ИИ (Minimax + Alpha-Beta Pruning)

Курсовая работа по объектно-ориентированному программированию  
Тема №5: Моделирование игры «Крестики-нолики» на поле произвольного размера с ботом.

##  Возможности
- Игра **человек против ИИ**
- Поддержка поля **n×n** (n ≥ 3)
- Победа при **k подряд** (k ≤ n)
- Алгоритм **минимакс с альфа-бета отсечением**
- **Демонстрация отсечённых ветвей** (лог количества пропущенных узлов)

##  Архитектура
- `GameBoard` — состояние игрового поля  
- `Player` (абстрактный) → `HumanPlayer`, `AIPlayer`  
- `Game` — управление ходами и проверкой завершения игры
